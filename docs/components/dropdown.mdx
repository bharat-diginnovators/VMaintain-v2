# Reusable Dropdown Component

The `Dropdown` component is a customizable React component for creating dropdown menus with various styling and behavioral options. It's suitable for selecting from a list of options in forms or other interactive elements.

## Component Code

```jsx
import React, { useState, useEffect, useRef } from "react";
import down from "../assets/Icons/down.svg"

const Dropdown = ({
  options = [],
  selected = "",
  onSelect = () => {},
  width = "w-full", 
  placeholder = "Select an option",
  menuWidth = "", 
  bgColor = "bg-[#F7F7F8]",
  borderColor = "border-gray-200",
  focusColor = "focus:ring-[#882EFD] focus:border-[#882EFD]",
  textColor = "text-left",
  customClass = "",
}) => {
  const [isOpen, setIsOpen] = useState(false);
  const [currentSelection, setCurrentSelection] = useState(selected);
  const dropdownRef = useRef(null);

  const toggleDropdown = () => {
    setIsOpen(!isOpen);
  };

  const handleOptionClick = (option) => {
    setCurrentSelection(option);
    onSelect(option);
    setIsOpen(false);
  };

  const handleClickOutside = (event) => {
    if (dropdownRef.current && !dropdownRef.current.contains(event.target)) {
      setIsOpen(false);
    }
  };

  useEffect(() => {
    document.addEventListener("mousedown", handleClickOutside);
    return () => {
      document.removeEventListener("mousedown", handleClickOutside);
    };
  }, []);

  return (
    <div className={`  ${width} ${customClass}`} ref={dropdownRef}>
      <button
        onClick={toggleDropdown}
        className={`flex justify-between w-full  items-center px-3 py-1.5 ${bgColor} border ${borderColor} ${focusColor} sm:text-sm rounded-md ${textColor}`}
      >
        {currentSelection || placeholder || <spa></spa>}
        <img src={down} alt="Dropdown Arrow"  className="self-end"/>
      </button>
      {isOpen && (
        <ul
          className={`absolute z-10  mt-1 ${menuWidth || width} ${bgColor} border ${borderColor} rounded-md shadow-lg`}
        >
          {options.map((option, index) => (
            <li
              key={index}
              onClick={() => handleOptionClick(option)}
              className="cursor-pointer px-4 py-2 hover:bg-[#882EFD] hover:text-white md:text-sm lg:text-[1rem]"
            >
              {option}
            </li>
          ))}
        </ul>
      )}
    </div>
  );
};

export default Dropdown;
```

## Component Explanation

- **Props**:
  - `options`: An array of options to display in the dropdown menu.
  - `selected`: The currently selected option.
  - `onSelect`: Function to handle the selection of an option.
  - `width`: Width of the dropdown button, default is `w-full`.
  - `placeholder`: Placeholder text when no option is selected.
  - `menuWidth`: Width of the dropdown menu, if different from the button width.
  - `bgColor`: Background color of the dropdown button and menu.
  - `borderColor`: Border color of the dropdown button and menu.
  - `focusColor`: Focus state styling for the dropdown button.
  - `textColor`: Text alignment and color for the dropdown button.
  - `customClass`: Additional custom CSS classes.

- **Functionality**:
  - The component toggles the dropdown menu when the button is clicked.
  - The `handleOptionClick` function updates the selected option and closes the dropdown.
  - The component listens for clicks outside the dropdown to close the menu.

## Usage Example

Below is an example of how to use the `Dropdown` component in your MDX file:

```mdx
<Dropdown
  options={["Option 1", "Option 2", "Option 3"]}
  selected=""
  onSelect={(option) => console.log(option)}
  width="w-64"
  placeholder="Choose an option"
  bgColor="bg-white"
  borderColor="border-gray-300"
  focusColor="focus:ring-blue-500 focus:border-blue-500"
  textColor="text-black"
  customClass="my-4"
/>
```
